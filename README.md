第一次作业说明文档

框架名称：commandapp

包含：greetings/hello，say/goodbye，course_selects三个主要界面

其中greetings/hello功能有：

  1.link到国科大主页
   
  2.显示本地时间
   
  3.实现与saygoodbye页面的跳转
   
  4.link到course_selects界面，查看个人选课信息
   
 say/goodbye功能

   跳转到greetinghello界面
   
course_select功能

   课程的添加，查看，修改，删除
   
数据库表名称：CourseSelect

包含属性有：课程编号，课程名称，学分，授课教师
